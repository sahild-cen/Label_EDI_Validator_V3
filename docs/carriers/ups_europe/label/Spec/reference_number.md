# Field: reference_number

## Display Name
Reference Number

## Group Description
Customer-generated unique package identifier embedded in positions 11-17 of the Tracking Number. Required to be unique for each package.

## Sub-Fields

### reference_number
- **Data Type:** alphanumeric
- **Length:** 7 (positions 11-17 of tracking number)
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted (customer-generated, must be unique per package)
- **Required:** yes (must be unique for each package)
- **Description:** Positions 11-17 of the Tracking Number, generated by the customer. Similar to a progressive number used in the transportation industry. Each package must have a unique reference number.
- **Detect By:** Extracted from tracking number positions 11-17
- **Position on Label:** Embedded within tracking number
- **ZPL Font:** Not specified
- **Field Prefix:** None
- **ZPL Command:** Part of tracking number ^FD
