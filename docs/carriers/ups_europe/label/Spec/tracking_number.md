# Field: tracking_number

## Display Name
UPS Tracking Number

## Group Description
The UPS customized unique package identifier used for tracking/tracing services. It is encoded in Code 128 barcode and printed as human-readable text on the label.

## Sub-Fields

### tracking_number
- **Data Type:** alphanumeric
- **Length:** Not fully specified in this extract; references positions 3-8 (account number), 9-10 (service indicator), 11-17 (reference number), plus check digit
- **Pattern/Regex:** Not specified in spec (this extract); starts with "1Z" per UPS convention referenced in definitions
- **Allowed Values:** Not restricted (generated by UPS system)
- **Required:** yes
- **Description:** The full UPS tracking number. Contains embedded fields: positions 3-8 = UPS Account Number, positions 9-10 = Service Indicator, positions 11-17 = Reference Number (unique per package), final character = Modified MOD 10 check digit.
- **Detect By:** barcode_data:^1Z, text_prefix:1Z
- **Position on Label:** Not specified in this extract
- **ZPL Font:** Not specified in this extract
- **Field Prefix:** None
- **ZPL Command:** ^FD (text field)

## Edge Cases & Notes
- The Reference Number (positions 11-17) is required to be unique for each package.
- The check digit uses Modified MOD 10 algorithm.
- The UPS Account Number (positions 3-8) is the unique number assigned by UPS to the shipper.
- The Service Indicator (positions 9-10) indicates the package's particular service features.
