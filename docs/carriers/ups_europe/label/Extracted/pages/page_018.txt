UPS CONTROL NUMBER (UPS Ready™ vendors)
The control number must print in the bottom right 
corner of all labels. The font size should be as small 
as possible while still being legible. The system 
control number will serve three purposes:
1. 
Will differentiate between certified UPS Ready™ 
vendors.
2. 
Will identify the version of software that 
produced the label.
3. 
Will identify the printer that generated the label.
