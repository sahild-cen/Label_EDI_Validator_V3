Barcode Specifications — Code 128
Barcode size based on Printer DPI
The 1Z barcode utilizes Code 128 symbology that 
can shift from subset A (used for alphanumeric) 
to subset C (numeric only).  As illustrated in the 
diagram below, each character in Subset A is 
counted as 1 and in Subset C each pair of numbers 
is counted as 1.  All combined, the total number of 
characters is15.
