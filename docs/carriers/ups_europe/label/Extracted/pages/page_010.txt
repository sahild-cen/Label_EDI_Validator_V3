PACKAGE COUNT
The package count indicates the number of this 
package N, related to the number of packages in
the entire shipment X (For Example: 1 OF 5 where
N = 1 and X = 5). The package count must print in the 
top right corner of the label. For a multi shipment - if 
the total number of packages is not known when 
the labels are printed, the package count should be 
displayed as 1 OF __ or 2 OF __, or 3 OF__.

UPS SHIPMENT NUMBER
For international shipments, the shipment number 
must print on all packages within the shipment, and 
is calculated from the tracking number of the first 
package in the ship­
ment. This number must appear
