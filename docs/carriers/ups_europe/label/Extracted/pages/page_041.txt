Additional Routing Instructions Block
Documentation indicators are required for all shipments generating in Europe. These indicators must print 
right-justified in the Additional Routing Instructions Block immediately below the highlight bar of the Tracking 
Number Barcode Block. The font size is 16 pt. bold.

PDF417 Barcode
A PDF417 Barcode is required for certain movements. For details on when the PDF417 is required and how 
to generate the barcode, please refer to the PDF417 Barcode Symbology Guide to Labeling Supplement.
Movement Reference Number
When a shipper files electronically and receives or provides a movement reference number for a shipment, the text 
shown below must print left-justified beneath the General Description of Goods. 
The movement reference number must always take precedence over the printing of other reference numbers 
on the address label.
