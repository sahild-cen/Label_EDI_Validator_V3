# Field: shipper_state_province

## Display Name
Shipper State/Province

## Field Description
The state, province, or region of the shipment origin/sender.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable, typically 2-35 characters
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted (standard state/province codes or names)
- **Required:** conditional — required where applicable (e.g., US, CA, AU)
