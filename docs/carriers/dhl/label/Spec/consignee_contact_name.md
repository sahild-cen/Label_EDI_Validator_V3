# Field: consignee_contact_name

## Display Name
Consignee Contact Name

## Field Description
The name of the specific contact person at the destination, especially when the consignee is a company.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable, up to 35 characters
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** conditional — recommended when consignee is a company

## Examples from Spec
No examples in spec.

## Position on Label
Within the consignee address block, typically as an "ATTN:" or contact line.

## Edge Cases & Notes
Useful for ensuring delivery to the correct individual within a business.

## Claude Confidence
MEDIUM — Common but specific requirement level varies.
