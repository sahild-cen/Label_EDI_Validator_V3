# Field: service_icon

## Display Name
Service Icon / Service Indicator

## Field Description
A visual icon or indicator on the label that represents the DHL service type, enabling quick visual identification of service level by handlers.

## Format & Validation Rules
- **Data Type:** string (icon/graphic identifier)
- **Length:** Not applicable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** DHL-defined service icons (e.g., Express Worldwide icon, Express 9:00 icon, Express 12:00 icon)
- **Required:** yes

## Examples from Spec
No examples in spec.

## Position on Label
Typically in the upper-left or upper-right corner of the label.

## Edge Cases & Notes
DHL uses specific colored/styled icons for different service tiers. The DHL Express logo and service-level branding are typically pre-printed or dynamically generated elements.

## Claude Confidence
MEDIUM — Standard visual element on DHL labels but specific icon definitions not in extracted text.
