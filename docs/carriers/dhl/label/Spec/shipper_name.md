# Field: shipper_name

## Display Name
Shipper Name (From Name)

## Field Description
The name of the person or company sending the shipment.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable, typically up to 35-45 characters
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** yes

## Examples from Spec
No examples in spec.

## Position on Label
In the shipper/origin address block, typically in the left portion of the label.

## Edge Cases & Notes
May include both company name and contact person name as separate lines.

## Claude Confidence
HIGH — Standard address element on all shipping labels.
