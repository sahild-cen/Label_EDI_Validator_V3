# Field: reference_number_2

## Display Name
Reference Number 2 (Additional Reference)

## Field Description
An additional customer-defined reference number for the shipment.

## Format & Validation Rules
- **Data Type:** alphanumeric
- **Length:** variable, typically up to 35 characters
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted (customer-defined)
- **Required:** no

## Examples from Spec
No examples in spec.

## Position on Label
In the reference/details section of the label, below the primary reference.

## Edge Cases & Notes
Not all label formats support a second reference line.

## Claude Confidence
MEDIUM — Common but support varies by label format.
