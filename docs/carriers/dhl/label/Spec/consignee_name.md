# Field: consignee_name

## Display Name
Consignee Name (To Name / Receiver Name)

## Field Description
The name of the person or company to whom the shipment is being delivered.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable, typically up to 35-45 characters
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** yes

## Examples from Spec
No examples in spec.

## Position on Label
In the consignee/destination address block, typically in the center or right portion of the label, prominently displayed.

## Edge Cases & Notes
May include both company name and attention/contact person name as separate lines. This is typically the largest address block on the label.

## Claude Confidence
HIGH — Standard required element on all DHL labels.
