# Field: shipper_phone

## Display Name
Shipper Phone Number

## Field Description
The telephone number of the shipper/sender for contact purposes.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** conditional — recommended, may be required for certain services or destinations

## Examples from Spec
No examples in spec.

## Position on Label
Within or near the shipper address block.

## Edge Cases & Notes
Should include country dialing code for international shipments.

## Claude Confidence
MEDIUM — Common on DHL labels but requirement level varies by service.
