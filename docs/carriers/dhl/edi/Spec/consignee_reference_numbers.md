# Field: consignee_reference_numbers

## Display Name
Consignee / Party Reference Numbers (VAT)

## Field Description
A segment (SG47 - RFF) indicating the Reference Numbers (i.e. VAT number) of different party roles such as Consignee, Importer, or Exporter.

## Format & Validation Rules
- **Data Type:** alphanumeric
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** conditional
