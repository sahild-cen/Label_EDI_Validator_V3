# Field: package_identification

## Display Name
Package Identification / Markings

## Field Description
A segment (SG55 - PCI) to indicate the type of markings on the item at piece level.

## Format & Validation Rules
- **Data Type:** alphanumeric
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** yes (Required at SG55, 1 occurrence of PCI)

## Examples from Spec
No examples in spec.

## Position on Label
Not specified in spec.

## Edge Cases & Notes
PCI is accompanied by GIN (Goods Identity Number) which contains the bar coded identification code.

## Claude Confidence
MEDIUM — Described in piece-level structure.
