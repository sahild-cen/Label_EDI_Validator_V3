Note 1496: 
If data in SG52/MEA and SG53/DIM is not available on item level, the GID should be 
used as follows: 
Element 1496 contains value 99999 and element 7224 contains the totals number of 
pieces in the shipment. The segment SG52 MEA and SG53 DIM will then contents the 
total volume. 
Note 7224: 
The number of packages is always 1, since the following segments indicate items within 
one shipment. 
BGM is on shipment level, within this group SG50, GID is on piece level. Since the GID 
group may occur 999 times, a shipment can contain up to 999 pieces.
