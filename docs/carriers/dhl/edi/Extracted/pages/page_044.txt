Note 3227: 
Use: ‘5’ - Place of origin. Use for DHL Express TD  
 
Note 3225: 
To be populated with DHL Express Service Area code for the location of the shipment 
origin. A list of DHL Express Service Area codes with corresponding DHL Express 
Facility codes by location name, post code and country is available on request. 
 
Note 3055: 
Use: ‘6’ – UN/ECE  
 
Note 3223: 
To be populated with DHL Express Facility code for the location of the shipment origin. 
May or may not be the same value as in 3225. A list of DHL Express Service Area codes 
with corresponding DHL Express Facility codes by location name, post code and country 
is available on request.
