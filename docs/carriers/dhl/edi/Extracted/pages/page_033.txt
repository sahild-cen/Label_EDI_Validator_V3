Example: 
COM+31 653 123 456:TE’ 
COM+?+31 653 123 456:TE’ 
 
Note: 
This segment is used to indicate a communication contact with the Shipper or Consignor 
of the shipment. 
 
Note 3148: 
If the phone number is beginning with the ‘+’ sign, the ‘?’ release character should come 
immediately before the ‘+’ sign in order to avoid translation problem. 
Only 70 characters will be accepted by DHL. 
 
Note 3155: 
Use: 
 
 
‘EM’ - Electronic Mail. 
‘FX’ – Telefax. 
‘TE’ - Telephone. 
‘AL’ - Cellular phone (Mobile)
