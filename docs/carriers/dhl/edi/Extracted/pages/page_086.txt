Note: 
 
This segment may be used to record various reference numbers for each line items.   
Values are validated against Shipper OR Consignee Country Codes under SG4-NAD-
3207 and SG43-NAD-3207. 
 
Note 1153: 
Use: 
 
‘HS’ - Line Item Harmonized code 
‘MF’ - Manufacturer's part number 
‘AFE’ – Export Control Commodity number (ECCN) 
For the full list of Line Item Reference Numbers type codes, refer to "DHL Express 
Integrated Solution Reference Information" (available from DHL Express on request). 
 
Note 1154: 
Only 35 characters will be accepted by DHL.
