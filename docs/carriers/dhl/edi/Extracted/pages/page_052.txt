Example:

DHL Day Definite: LOC+7+AMS+UTX’ 
 
 
Note: 
The segment LOC with qualifier ‘8’ is mandatory to indicate the DHL Express Service 
Area code and DHL Express Facility code corresponding to the destination location of the 
shipment. 
It is also mandatory for DHL Express Road shipments with product feature ‘Hold at 
Depot’ and ‘PackStation’. In this segment, the appropriate code is specified. 
 
Note 3227: 
Use: 
‘8’ - Place of destination. Use for DHL Express TD 
‘7’ - Place of delivery. Use for DHL Express DD       
 
Note 3225: 
To be populated with DHL Express Facility code for the location of the shipment 
destination. May or may not be the same value as in 3223. A list of DHL Express Service 
Area codes with corresponding DHL Express Facility codes by location name, post code 
and country is available on request. 
 
Note 3223:

destination. May or may not be the same value as in 3225. A list of DHL Express Service
