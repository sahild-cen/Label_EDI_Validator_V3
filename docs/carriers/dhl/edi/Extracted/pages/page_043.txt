Note 4052:1 
This element is reserved for product features. All appropriate product features should be 
stored in this element, without separation sign. See example. The format of each 
occurrence is 3 digits length, leading ‘0’ (zero) included. Please contact your Sales 
representative for valid product features codes.  
The length of this element (AN..70) can handle a maximum of 23 product features codes 
(Extra Charge Code). Extra Charge Codes with related Account Numbers should always 
be stored in this element first followed by the Extra Charge Codes without related 
Account Number. 
 
Note 4215:         Use ‘A’ – To Indicate that the charges are to be charged to an account. 
 
Note 4052:2: 
Account Numbers (to which the extra charges will be applied) should be stored in this 
element, without separation sign, and must be in the same order as the related to an 
Extra Charge Code are placed in element 4052:1. See example. The format of each 
occurrence is 9 digits length. The length of this element (AN..70) can handle a maximum 
of 7 Account Numbers.
