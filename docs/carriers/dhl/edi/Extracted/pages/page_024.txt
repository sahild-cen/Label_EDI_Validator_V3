Note 0068: 
DHL Express determined identification for recognizing the name of the source shipping 
system platform, application or software that created the message. The data is divided with 
Slash (‘/’). Value GM2 is to be given before the slash. The name of the source shipping 
platform is given after the slash. The source name must be bilaterally agreed by the party 
generating the message (so the DHL Express customer or their shipping system provider) 
and DHL Express. For instance this could be a concatenation of the name, version and 
release of the generating software.
