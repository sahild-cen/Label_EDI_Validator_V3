Note 5004: 
Rules for numeric value: 
▪ 
The maximum number of digit is 18 where the 3 decimals are included. 
▪ 
All amounts have to be specified with 3 decimals included, regardless currency. 
▪ 
Decimal sign depends on what has been specified in segment UNA.  
If UNA is not specified then default sign will be point. 
▪ 
COD value has to be given in the currency of the destination country. 
 
Note 6345: 
Currency codes are according to ISO 4217.  
This must be the currency of the shipper country except for qualifier ‘22’ – COD currency 
of destination needs to be populated.
