Note 4451: 
Use: 
‘LIN’ – Line Item indicator for GID 
‘AAA’ - Line Item Description 
For low value shipments to Singapore Y or N must be given in SG50 within the GID 
segment - this is to indicate whether GST on the shipment contents has been paid in the 
exporting country already (Y = yes) or not (N = no), (see also SG-6 RFF) 
Examples: 
FTX+Y' 
FTX+N' 
Each line item should have such a GST flag/ indicator.
