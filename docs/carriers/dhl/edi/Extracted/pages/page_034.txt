Function: To identify a reference, in this case the VAT number and Registration numbers, used by a party.  
This segment group may be used for customs clearance shipments (only destinations in non-EU countries).

RFF+ZZZ:123456789:SG 
 
Note: 
This segment used to record the Registration numbers of the different role types.  For 
example, VAT number of the Shipper/Consignor, Route Code and Cycle code.  
 
Values are validated against role types under the DHL equivalent values of SG4-NAD-
3035. 
 
Note 1153: 
For the full list of possible Registration Number Type for different role types / Country, 
refer to "DHL Express Integrated Solution Reference Information" (available from DHL 
Express on request) 
 
Examples include: 
‘VA’ – VAT Registration Number 
                          ‘AUS’ – Delivery Route reference or shipper route code and pickup cycle code  
‘ZZZ’ – Overseas Registered Supplier Reference No.”, for example where it is required 
with low value shipments to Singapore (see also SG-50 FTX)
