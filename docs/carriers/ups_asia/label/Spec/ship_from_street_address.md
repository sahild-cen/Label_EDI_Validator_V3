# Field: ship_from_street_address

## Display Name
Ship From Street Address

## Field Description
The street address of the origin/shipper location.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** All uppercase characters
- **Required:** yes

## Examples from Spec
- `STREET ADDRESS`

## Position on Label
In the Ship From address block, must print directly above the Postal Code line.

## Edge Cases & Notes
- Font Size = 8 pt.
- Standard abbreviations should be employed

## Claude Confidence
HIGH — clearly shown in label samples
