# Field: ship_to_postal_code_city

## Display Name
Ship To Postal Code / City Line

## Field Description
The postal code and city (and state/province if applicable) for the consignee/recipient address.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** yes

## Examples from Spec

## Position on Label
Ship To address block, below the street address.
