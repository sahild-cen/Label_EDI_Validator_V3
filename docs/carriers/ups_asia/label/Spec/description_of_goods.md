# Field: description_of_goods

## Display Name
Description of Goods (DESC)

## Field Description
A brief text description of the contents/goods being shipped in the package.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** yes

## Examples from Spec
- `DESC: AUTO PARTS`
- `DESC: CLOTHING`
- `DESC: MARBLE`
- `DESC: TOOLS`
- `DESC: PRINTER`
- `DESC: COMPUTER EQUIPMENT`
- `DESC: CIRCUIT BOARDS`

## Position on Label
In the lower portion of the label, prefixed with "DESC:".

## Edge Cases & Notes
Always prefixed with "DESC:" on the label. Important for customs and shipment identification purposes.

## Claude Confidence
HIGH — Consistently shown on all label examples with the same prefix format.
