# Field: highlighting_bars

## Display Name
Highlighting Bars

## Field Description
One-tenth inch solid black lines located immediately above and below the UPS Barcode Block, used to visually delimit the barcode area.

## Format & Validation Rules
- **Data Type:** graphic element
- **Length:** N/A
- **Pattern/Regex:** N/A
- **Allowed Values:** Solid black lines, 0.1 inch thick
- **Required:** yes

## Examples from Spec
No examples in spec.

## Position on Label
Immediately above and below the UPS Barcode Block.

## Edge Cases & Notes
- These are structural elements that frame the barcode block on the label.
- Must be solid black and exactly one-tenth inch (0.1") in height.
