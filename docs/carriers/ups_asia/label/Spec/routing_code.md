# Field: routing_code

## Display Name
Routing Code

## Field Description
The UPS routing code that identifies the destination sort facility, displayed in a prominent position on the label. Composed of a 3-letter country code, a 3-digit postal prefix, and a sort code segment.

## Format & Validation Rules
- **Data Type:** alphanumeric
- **Length:** variable (approximately 12 characters)
- **Pattern/Regex:** `[A-Z]{3}\s+\d{3}\s+\d-\d{2}`
- **Allowed Values:** Not restricted — generated by UPS systems
- **Required:** yes

## Examples from Spec
"DEU 063 9-39", "SGP 256 2-00", "FRA 753 7-00", "AZ 852 1-21", "JPN 106 1-00", "IL 606 9-02", "GA 301 9-01", "GA 300 9-05", "JPN 695 2-00", "IND 092 1-00", "CHN 710 0-00", "DEU 091 0-00"

## Position on Label
Middle section of the label, prominently displayed. Appears both as human-readable text and encoded in the MaxiCode/barcode. Appears twice — once in the routing section and once near the service icon area.

## Edge Cases & Notes
For US domestic destinations, the routing code uses the 2-letter state abbreviation instead of a 3-letter country code (e.g., "AZ 852 1-21", "IL 606 9-02", "GA 301 9-01"). For international destinations, the 3-letter ISO country code is used (e.g., "DEU", "SGP", "FRA", "JPN", "IND", "CHN").

## Claude Confidence
HIGH — field appears consistently across all label examples with clear pattern variations for domestic vs. international
