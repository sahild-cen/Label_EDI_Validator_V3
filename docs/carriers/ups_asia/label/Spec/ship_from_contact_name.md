# Field: ship_from_contact_name

## Display Name
Ship From Contact Name

## Field Description
The contact name for the shipper/sender of the package, displayed in the ship-from address block at the top of the label.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** yes

## Examples from Spec
"CONTACT NAME"

## Position on Label
Upper left portion of the label, first line of the ship-from address block.

## Edge Cases & Notes
Appears consistently across all label examples as the first element in the shipper section.

## Claude Confidence
HIGH — field appears consistently across all label examples in the spec
