# Field: barcode_block

## Display Name
UPS Barcode Block

## Field Description
A standardized unit of area on the label in which bar-coded information is presented. Barcode blocks are typically stacked vertically and separated from each other by a horizontal line.

## Format & Validation Rules
- **Data Type:** graphic/structural element
- **Length:** N/A
- **Pattern/Regex:** N/A
- **Allowed Values:** Contains barcode symbols with highlighting bars above and below
- **Required:** yes

## Examples from Spec
No examples in spec.

## Position on Label
Central area of the label; blocks are stacked vertically with separating lines between them.

## Edge Cases & Notes
- Multiple barcode blocks may appear on a single label, stacked vertically.
- Separated by horizontal separating lines (narrow lines used to segregate barcode and information blocks).
