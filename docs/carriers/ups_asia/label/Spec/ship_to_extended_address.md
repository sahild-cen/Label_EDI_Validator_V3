# Field: ship_to_extended_address

## Display Name
Ship To Extended Address

## Field Description
An additional address line for the consignee/recipient (e.g., suite, floor, building name).

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** no — optional additional address line

## Examples from Spec
"EXTENDED ADDRESS"

## Position on Label
Middle left portion of the label, in the ship-to address block below company name.

## Edge Cases & Notes
Not all label examples include this field; it is optional.

## Claude Confidence
HIGH — field appears in many but not all label examples
