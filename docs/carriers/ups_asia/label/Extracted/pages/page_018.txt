Additional Routing Instructions Block
LOCATION
Additional Routing Instructions are located beneath the 
bottom Tracking Number highlight bar, left-justified using all 
uppercase characters. 
SIGNATURE REQUIRED ACCESSORIALS
When UPS Delivery Confirmation Signature Required 
accessorials are selected, the routing instruction must print 
left-justified immediately below the Billing Option line. The 
font size is 10 pt.
SIGNATURE REQUIRED
BILLING OPTION
The billing option must print directly below 
the highlight bar, at the top of this section.
