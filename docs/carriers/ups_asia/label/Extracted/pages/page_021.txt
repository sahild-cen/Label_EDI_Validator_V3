UPS PRINT RETURN LABEL FORMAT — 
INTERNATIONAL MOVEMENTS
• KEY-RS
When Print Return Label is selected as the return 
service option:
The following two indicators are only for goods 
shipped within the EU or from the EU to the rest 
of the world. The font size is 15 pt. bold.
The text RS must appear in the Additional Routing 
Instructions Block. The text must print with the 
Documentation Indicator.
• 
For Example:  
KEY-RS
Shipping System Generates Invoice for EU Goods 
Not in Free Circulation 
• Font Size                       = 20 pt. bold
• INV/T1/RS
Shipping System Does Not Generate Invoice 
for EU Goods Not in Free Circulation
• KEY/T1/RS
The text DESC: followed by a description of the 
merchandise must appear in the Additional Routing 
Instructions Block immediately below the Billing 
Option line.
• Font Size                    = 10 pt.
• For Example:  DESC: CLOTHING
The text RETURN SERVICE-PAYMENT GUARANTEED 
must appear in the Additional Routing Instructions 
Block immediately below the Description line.
• 
Routing Instruction = RETURN SERVICE-PAYMENT 
GUARANTEED
• Font Size                     = 10 pt.
DOCUMENTATION INDICATORS FOR UPS RETURNS
Documentation indicators combined with the RS 
indicator must be used for UPS Returns as noted. 
These indicators must print right-justified in the 
Additional Routing Instructions Block immediately 
below the highlight bar of the Tracking Number 
Barcode Block. The font size is 16 pt. bold, unless 
otherwise indicated.
