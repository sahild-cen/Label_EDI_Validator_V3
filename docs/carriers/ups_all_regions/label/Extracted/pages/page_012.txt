PACKAGE COUNT 
The package count indicates the number of the 
package N, related to the number of packages 
in the entire shipment X (e.g., N of X). Only N will 
print on the over-label except, when the shipment 
is the Document Box. The complete N of X for the 
consolidated shipment must print only on the 
document box.
DOCUMENT BOX 
This section contains the final destination 
information for the shipment. All data formatting 
and placement of the information follows the 
current UPS Guide to Labeling. In addition, an 
indicator must print directly to the right (preceded 
by a dash) of the documentation indicator in 
the top right corner of the Additional Routing 
Instructions Block. All labels including the 
document box will display the same indicator.
