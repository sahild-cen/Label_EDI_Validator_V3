Additional Routing Instructions Block
The package pallet label is produced for the UPS 
service provider to help identify loose packages that 
will be processed through the regular UPS system. 
These packages are segregated on a separate pallet 
and identified with this label. 
Package Information Block 
The Additional Routing Instructions block contains 
piece counts and reference numbers.

PALLET COUNT
The pallet count indicates the number of this pallet 
N, related to the total number of package pallets in 
the entire shipment X (e.g., N of X). The pallet count 
appears on the first line in the upper right corner of 
the label.
