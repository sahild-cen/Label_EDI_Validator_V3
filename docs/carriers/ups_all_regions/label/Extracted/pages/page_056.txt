Dry Ice and Biological Substances
Label Format Requirements
Overview
Dry Ice and Biological Substances are offered as 
contract services.  When using these services, the 
label must contain additional information to meet 
regulatory requirements and assist UPS in processing 
the shipment.
Additionally, an Audit Indicator line must print below 
the Dry Ice line indicating whether an audit of the 
package is necessary by UPS Operations. For more 
details on which packages will require an audit, 
please refer to the UPS Rate and Service Guide.
For packages which require audit:
• Routing Instruction = AUDIT REQUIRED
NOTE:  Dry ice is a general service offering for 
certain United States and Puerto Rico domestic 
shipments.
For packages which do not require audit:
• Routing Instruction = NO AUDIT REQUIRED
• Font Size = 10 pt.
The service may be used with UPS Returns™ or UPS 
Import Control™.  The label requirements can be 
found in the main UPS Guide to Labeling document.
