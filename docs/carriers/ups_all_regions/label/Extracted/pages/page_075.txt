In order to receive a block of numbers, you will need 
to submit a block of 20 numbers beginning with 
1111111111 for verification of check digit calculation.  
Your system is required to configure such that the 
currently issued block of numbers cannot be reused.  It 
is recommended that a low pool warning be displayed 
when the block is close to being exhausted.  The 
user must be prevented from shipping if the block of 
numbers has been exhausted.  To obtain validation of 
your system generated Consolidated Movement (CM) 
Shipment ID numbers, please contact your Trade Direct 
Account Executive.

The system code is a two character alphanumeric 
identification code that identifies the type of system 
used to generate the Sub Pro number.  This two character 
code will remain numeric until all numeric possibilities 
are exhausted. Host customers and UPS Ready Vendors 
must use "09".
