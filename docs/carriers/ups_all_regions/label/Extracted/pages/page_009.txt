PACKAGE COUNT 
The package count indicates the number of the 
package N, related to the number of packages 
in the entire shipment X (e.g., N of X). Only N will 
print on the over-label except, when the shipment 
is the Document Box. The complete N of X for the 
consolidated shipment must print only on the 
document box.

INSTRUCTION TEXT
This block contains instructions for the customer 
and UPS Operations. The customer text explains 
to the customer how to apply the over-label to 
the package. This text will be translated when 
appropriate. The UPS text informs Operations 
when to remove the over-label. The over-label 
will be removed at the import site. The UPS text 
must always be in English.
