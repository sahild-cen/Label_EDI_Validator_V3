The label is created at a UPS facility - so creating a 
label is not required, but the service indicator will 
need to be transmitted to UPS.
The UPS Returns™ Pack and Collect service is a 
contractual returns service for the retrieval of high-
value products from the original receiver for return 
to the shipper or to an alternate location.  The UPS 
Returns™ Pack and Collect service allows a shipper 
to initiate a next day collection leveraging UPS 
Return Services RS1 and RS3 and a selection of 
specially designed, UPS-branded UPS Returns Pack 
and Collect packaging. 
The UPS service provider performs a basic visual 
inspection of the item to be returned and it is 
packed in the empty UPS-provided packaging.
