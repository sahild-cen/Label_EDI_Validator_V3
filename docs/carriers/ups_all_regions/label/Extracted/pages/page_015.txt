TRACKING NUMBER
The last three digits of the shipment’s virtual 
document box 1Z tracking number will print on the 
summary label

• For Example:  TRKG #: 123

GLOBAL CUSTOMS CLEARANCE NUMBER (GCC#)
The shipment’s virtual document box 1Z tracking 
number will be used to calculate the GCC# which is 
the same as the Shipment Number.  See Appendix D 
of the UPS Guide to Labeling.
• 
