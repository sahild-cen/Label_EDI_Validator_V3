# Field: ship_from_company_name

## Display Name
Ship From Company Name

## Field Description
The name of the company or business from which the shipment originates.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** yes

## Examples from Spec
- `COMPANY NAME` (placeholder used throughout spec)
- `WORLDEASE US SHIPPER` (World Ease example)

## Position on Label
Top section of the label in the ship-from address block, below the contact name and phone number.

## Edge Cases & Notes
For World Ease shipments, the shipper name may indicate "WORLDEASE US SHIPPER" or similar designation.

## Claude Confidence
HIGH — Consistently present in all label examples.
