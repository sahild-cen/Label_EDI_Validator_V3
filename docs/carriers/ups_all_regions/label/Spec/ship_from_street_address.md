# Field: ship_from_street_address

## Display Name
Ship From Street Address

## Field Description
The street address of the shipping origin/sender location.

## Format & Validation Rules
- **Data Type:** string
- **Length:** variable
- **Pattern/Regex:** Not specified in spec
- **Allowed Values:** Not restricted
- **Required:** yes

## Examples from Spec
- `STREET ADDRESS` (placeholder used throughout spec)
- `UPS WAY`

## Position on Label
Top section of the label in the ship-from address block.

## Edge Cases & Notes
All spec examples use placeholder text. In actual use, this would contain the full street address.
