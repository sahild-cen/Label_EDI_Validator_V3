Segment Remarks:
Air Notes
For Air express products it is a combination of value in TDT and TSR segment to be used.
Code Z09 in combination with Denmark as sending country has special handling and ending up in application Centiro.
If shipments are handled as Air freight following codes are used. (ediEnterprise application)
DSV Priority                   Delivery same day   TSR++Z09+1'   TDT+20++3'
DSV Classic (Xpress)       Xpress Air              TSR++Z09+2'   TDT+20++4'
DSV Classic (Xpress)       Xpress Document   TSR++Z09+2'   TDT+20++5'
DSV Budget (Economy)                                TSR++Z09+3'   TDT+20++3'
DSV Parcels                                                TSR++Z02+3'   TDT+20++3'
No = Consecutive segment number
