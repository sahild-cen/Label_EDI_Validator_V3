Road Notes
Gross Weight at Shipment level (CNT+7) or for each line item (SG21 MEA) is required.
Number of packages at Shipment level (CNT+11) or for each line item (SG19 GID) is required.
At least one of the following is required:
Shipment level Volume (CNT+15)
Shipment level Loading Meters (CNT+57)
Each line item volume (SG21 MEA+VOL)
Each line item loading meters (SG21 MEA+LMT) 
Each line item dimensions (SG22 DIM)
